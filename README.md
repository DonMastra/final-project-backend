# final-project-backend
Spring Boot backend project for the final presentation of #YoProgramo

This is the RESTful API for the #YoProgramo portfolio final project frontend.

Technologies stack for this backend:

* Java 11
* Spring Boot 2.5
* Spring Data JPA
* Spring Security
* JWT
